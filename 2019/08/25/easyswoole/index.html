三、EasySwoole
1、简介

官方网站：https://www.easyswoole.com

​ EasySwoole 是一款基于Swoole Server 开发的常驻内存型的分布式PHP框架，专为API而生，摆脱传统PHP运行模式在进程唤起和文件加载上带来的性能损失。 EasySwoole 高度封装了 Swoole Server 而依旧维持 Swoole Server 原有特性，支持同时混合监听HTTP、自定义TCP、UDP协议，让开发者以最低的学习成本和精力编写出多进程，可异步，高可用的应用服务。

常用组件：

HTTP WEB 组件

TCP、UDP、WEB_SOCKET 组件

redis连接池

mysql 连接池

自定义进程

分布式跨平台RPC组件

微信公众号与小程序SDK

协程版微信、支付宝支付SDK

模板渲染引擎

Tracker链路跟踪

限流器

消息队列

协程HTTP客户端组件

apollo配置中心

validate验证器

验证码

fast-cache组件

Policy权限组件

IOC、协程上下文管理器

2、特性

强大的 TCP/UDP Server 框架，多线程，EventLoop，事件驱动，异步，Worker进程组，Task异步任务，毫秒定时器，SSL/TLS隧道加密。

EventLoop API，让用户可以直接操作底层的事件循环，将socket，stream，管道等Linux文件加入到事件循环中

定时器、协程对象池、HTTP/SOCK控制器、分布式微服务、RPC支持。

3、优势

简单易用开发效率高

并发百万TCP连接

TCP/UDP/UnixSock

支持异步/同步协程

支持多进程/多线程

CPU亲和性守护进程

4、环境要求

保证PHP版本大于等于 7.1

保证Swoole扩展版本大于等于4.4.0

需要pcntl扩展的任意版本

使用LInux/FreeBSD/MacOS 这三种操作系统

使用composer作为依赖包管理工具

5、安装 EasySwoole

注意：使用composer安装EasySwoole框架，得确保composer已经安装好。如果没有安装请求参考前面安装composer教程。

5.1、切换镜像源

composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/

5.2、安装

5.2.1、下载框架代码

shell># mkdir EasySwoole
shell># EasySwoole
shell># composer require easyswoole/easyswoole=3.x

有可能报以下错误：

以上错误代表着：有可能PHP扩展的zip扩展没有开启；Linxu操作没有安装unzip解压命令

解决方案：

shell># yum -y install unzip

5.2.2、安装

shell># php vendor/bin/easyswoole install

再然后编辑根目录下的 composer.json 文件，注册应用的命名空间，内容如下：

更新命名空间映射

composer dumpautoload

启动项目

php easyswoole start

5.4、安装IDE助手插件

由于 Swoole 的函数并不是 PHP 标准函数，IDE 无法进行自动补全，为了方便开发，需要安装IDE助手插件；在项目根目录下执行以下命令：

shell># composer require easyswoole/swoole-ide-helper

5.5、目录结构

EasySwoole 的目录结构是非常灵活的，基本上可以任意定制，没有太多的约束，但是仍然建议遵循下面的目录结构，方便开发

project 项目部署目录
├─App 应用目录(可以有多个)
│ ├─HttpController 控制器目录
│ │ └─Index.php 默认控制器
│ └─Model 模型文件目录
├─Log 日志文件目录
├─Temp 临时文件目录
├─vendor 第三方类库目录
├─composer.json Composer架构
├─composer.lock Composer锁定
├─EasySwooleEvent.php 框架全局事件
├─easyswoole 框架管理脚本
├─easyswoole.install 框架安装锁定文件
├─dev.php 开发配置文件
├─produce.php 生产配置文件

11.5.3、查询数据

EasySwoole\EasySwoole\App\HttpController\Api\Index.php

//查询数据
public function select()
{
//1、获取MySql对象
$db = Di::getInstance()->get(‘MYSQL’);
//获取所有数据
$data = $db->get(‘user’);
//实现一个分页
$page = 3;
$pagetotal = 2;
$dataPage = $db->get(‘user’, [($page-1)*$pagetotal, $pagetotal]);
//获取一条数据
$one = $db->getOne(‘user’);
return $this->writeJson(200, ‘OK’, [‘data’=>$data,’dataPage’=>$dataPage, ‘oen’=>$one]);
}
